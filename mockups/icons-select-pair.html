select × multi_select

两列各 10 个变种。每个左是 minimal（纯线条），右是 accent（terracotta 方块色）。 挑两个编号告我，比如 "select 03 + multi 07"。

select pick one from a list
01
Dropdown box — 圆角盒 + chevron-down，HTML <select>
02
Pill + 实心点 — 一个胶囊里有一个填充点
03
Radio button — 圆环 + 实心内圆，老派单选
04
Highlighted row — 三行里中间被高亮
05
Caret pointer — 三角指针指向一行
06
Combobox — 文本框 + 右侧 chevron 列
07
三 chip 一个填充 — 列表中间那个被选中
08
Circle + ✓ — 已选状态圆形带勾
09
Tag with chosen mark — 单 tag 带斜角勾
10
Pill + → arrow — 选中并跳转
multi_select pick many from a list
01
Ballot — 一行 4 个泡，两个填充（多选答题卡）
02
Venn — 两圆相交，集合"双归属"
03
Streamers — 一个结点 + 三条丝带（钥匙圈）
04
Overflow badge — 一个 chip + 右上"+2"角标
05
Constellation — 四点连线 Y 图，关联
06
Punch card — 3×3 网格，几个填充
07
Token row — 三大圆，奇数位填充
08
Box + 三点 — select 形状 + 内三选标
09
Cloud — 三 chip 不同位置散开
10
Path of picks — 一条折线穿过四个点
