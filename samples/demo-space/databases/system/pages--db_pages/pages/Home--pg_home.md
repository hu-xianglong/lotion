# Home

Welcome to the Lotion demo space.

This page tests Markdown rendering, embedded database views, editable table
cells, and formula fields.

## MVP Coverage

- Pages: open [Markdown Lab](databases/system/pages--db_pages/pages/Markdown_Lab--pg_markdown_lab.md), rename it,
  edit the text, and switch preview modes.
- Databases: open [Field Type Lab](databases/system/pages--db_pages/pages/Database_Lab--pg_database_lab.md) and
  edit every supported field type.
- Records: add a row, edit cells, then delete a row in
  [Tasks](databases/user/Tasks--db_tasks/pages/Add_a_formula_field--row_task_3.md).
- Views: open
  [Add a formula field](databases/user/Tasks--db_tasks/pages/Add_a_formula_field--row_task_3.md) for
  a real row page example.
- Search: try the [Search Lab](databases/system/pages--db_pages/pages/Search_Lab_Alpha_Invoice_Formula--pg_search_lab.md).
- Markdown: see every formatting surface in the
  [Markdown Showcase](databases/system/pages--db_pages/pages/Markdown_Showcase--pg_markdown_showcase.md).
- Backup: click the backup button after making a small edit.
